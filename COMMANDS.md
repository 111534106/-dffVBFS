# 🎵 Discord 音樂機器人指令手冊 (中文化版本)

目前機器人已全中文化，並加入自動重連功能。

## 🚀 核心指令

| 指令 | 說明 | 舊指令 (參考) |
| :--- | :--- | :--- |
| `/播放` | **播放音樂** (支援網址或關鍵字) | `/play` |
| `/跳過` | **跳過** 目前歌曲 | `/skip` |
| `/暫停` | **暫停** 播放 | `/pause` |
| `/繼續` | **繼續** 播放 | `/resume` |
| `/清單` | **查看播放清單** | `/queue` |
| `/停止` | **停止播放** 並離開頻道 | `/stop` |

---

## ✨ 新增功能：自動重連 (Auto-Reconnect)
- **連線守護**：如果因為網路波動導致機器人被迫離開頻道，它現在會自動嘗試重新加入。
- **長效播放**：針對 12 小時以上的影片進行了優化，確保連線穩定性。
